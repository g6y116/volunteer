# 봉사활동 앱

1365 자원봉사 사이트의 정보를 쉽게 조회할 수 있습니다.

플레이스토어에서 [다운로드](https://play.google.com/store/apps/details?id=g6y116.volunteer) 받으실 수 있습니다.

</div>

<br>
<br>

## Feature

- 봉사활동 조회(지역 / 시작일 / 종료일 / 봉사자 유형 / 모집 상태 / 신청 웹페이지로 이동)
- 책갈피 / 방문기록 / 다크모드 / 다국어 / 카카오 맵
